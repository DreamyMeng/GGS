### 当前项目使用插件：
- Newtonsoft.Json - 用于JSON数据的序列化和反序列化，方便存储游戏数据。  
com.unity.nuget.newtonsoft-json  


- Zenject - 是一个轻量级、高性能的依赖注入框架。  
https://github.com/DreamyMeng/GGS.git?path=Assets/Plugins/Zenject


- DOTween - 强大的动画插件，用于创建平滑的UI动画、角色动画等。  
https://github.com/DreamyMeng/GGS.git?path=Assets/Plugins/DOTween


- UniTask - 是一个为 Unity 量身定制的轻量级异步编程库。  
https://github.com/DreamyMeng/GGS.git?path=Assets/Plugins/UniTask


- GGS - 自定义框架，使用Zenject注入。  
UI - ui框架  
https://github.com/DreamyMeng/GGS.git?path=Assets/GGS/UI  
Data - 读取数据、玩家存档  
https://github.com/DreamyMeng/GGS.git?path=Assets/GGS/Data  
Localization - 多语言  
https://github.com/DreamyMeng/GGS.git?path=Assets/GGS/Localization
